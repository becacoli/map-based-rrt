## Objetivo  
Implementação do algoritmo **RRT (Rapidly-exploring Random Tree)**, **algoritmo de navegação baseada em mapa** utilizando o **ROS2 (Jazzy)** e o **simulador Gazebo**, para planejamento de trajetória em um ambiente com **passagem estreita e obstáculos fixos**.
